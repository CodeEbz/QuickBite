# QuickBite

A three-sided food delivery platform for customers, restaurants, and drivers.

## Structure

- `/mobile` — React Native + Expo (Android)
- `/web` — Next.js + Tailwind (Admin Dashboard)
- `/backend` — Spring Boot + PostgreSQL

## Deployment

- Mobile: Expo EAS Build
- Web: Vercel
- Backend + DB: Render

## Getting Started

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Mobile
```bash
cd mobile
npx expo start
```

### Web
```bash
cd web
npm run dev
```
